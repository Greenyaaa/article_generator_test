# Как запустить Article Generator

## Что нужно

- Python 3.10+
- Ключ от Anthropic API → [console.anthropic.com](https://console.anthropic.com/)

---

## Шаг 1 — Установить зависимости

```bash
pip install -r requirements.txt
```

## Шаг 2 — Добавить API ключ

Создай файл `.env` в папке проекта и вставь свой ключ:

```
ANTHROPIC_API_KEY=sk-ant-api03-ВАШ_КЛЮЧ_ЗДЕСЬ
```

## Шаг 3 — Создать входной файл

Создай файл `input.txt` (или используй готовый пример):

```
Topic: Почему малый бизнес должен автоматизировать рутинные процессы
Language: Russian
Bullets:
- ручной труд ограничивает рост
- найм сотрудников увеличивает расходы
- автоматизация улучшает масштабируемость
```

## Шаг 4 — Запустить

```bash
python main.py --input input.txt
```

После запуска в папке `output/` появятся два файла:
- `draft.md` — первая версия статьи
- `final.md` — улучшенная финальная версия

---

## Дополнительные параметры

| Параметр | Что делает | Пример |
|---|---|---|
| `--words` | Целевой размер статьи (по умолчанию 800) | `--words 1000` |
| `--model` | Модель Claude (по умолчанию claude-sonnet-4-5) | `--model claude-opus-4-5` |
| `--verbose` | Подробные логи в консоль | `--verbose` |

Пример с параметрами:

```bash
python main.py --input input.txt --words 600 --verbose
```

---

## Запустить тесты

```bash
pytest tests/ -v
```

Тесты не требуют API ключа — LLM замокан.
